# Back of House Strategies

A focused restaurant advisory practice. Single-file HTML prototype.

## Stack
- Pure HTML/CSS/JS (no build step)
- Google Fonts: Fraunces, Inter, JetBrains Mono
- Deployed via Vercel

## Local dev
Open `index.html` in any browser, or serve with:
```
python3 -m http.server 8000
```

## Deploy
Any push to `main` auto-deploys to Vercel.
